# Estética Automotiva - Batalha de Modelos (XML Prompt Engineering)

[cite_start]Este projeto consiste em uma Landing Page desenvolvida para o setor de estética automotiva, focada em carros esportivos[cite: 10]. [cite_start]O código foi gerado através de técnicas de **Engenharia de Prompt Estruturado em XML** como parte de uma atividade acadêmica de análise comparativa entre diferentes LLMs (Large Language Models)[cite: 1, 8].

## 🛠️ Tecnologias Utilizadas
* [cite_start]**HTML5/CSS3** (Single Page com CSS integrado)[cite: 8].
* [cite_start]**Estrutura XML** para definição de diretrizes de design e obrigatoriedades técnicas[cite: 10].
* [cite_start]**Modelos Avaliados:** ChatGPT, Gemini, DeepSeek, Qwen, Grok, Maritaca e Claude[cite: 9, 12].

## 🎨 Diretrizes do Projeto (Prompt XML)
[cite_start]O prompt estruturado definiu os seguintes parâmetros para a construção da página[cite: 10]:
* **Tema:** Estética automotiva com foco em carros esportivos.
* **Layout:** Responsivo e minimalista.
* **Paleta de Cores:** Laranja e preto.
* **Tipografia:** Sans-serif para títulos e Serif para o corpo.
* **Seções Obrigatórias:** Menu de navegação (âncoras), galeria de portfólio e rodapé de contato.

## 📊 Quadro de Análise Comparativa
[cite_start]Durante o experimento, foram coletadas métricas de precisão e eficiência de cada modelo[cite: 12, 13]:

| Critério | GPT | Gemini | DeepSeek | Qwen | Grok | Maritaca | Claude |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **Precisão do Prompt** | 95% | 90-95% | 100% | 98-100% | 94% | 100% | 92% |
| **Precisão do HTML** | 96% | 78% | 98% | 100% | 98% | 100% | 94% |
| **Tokens Gerados** | 750 | 1450 | 2850 | 1550 | 3059 | 1100 | 6000 |

## 🧠 Reflexão Crítica
* [cite_start]**Melhor Compreensão:** O modelo **Claude** demonstrou a melhor compreensão da estrutura XML proposta[cite: 17, 21].
* [cite_start]**Variação de Verbosidade:** Houve uma diferença significativa no consumo de recursos, variando de 750 a 6.000 tokens para o mesmo objetivo[cite: 19].
* [cite_start]**Conclusão:** Para prototipagem e códigos complexos, o Claude destacou-se pela criatividade e fidelidade às instruções[cite: 21].
