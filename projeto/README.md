# 🧠 Engenharia Reversa Assistida por IA

Projeto acadêmico desenvolvido com foco em engenharia reversa de aplicações web utilizando Inteligência Artificial Generativa.

---

## 🎯 Objetivo

- Realizar engenharia reversa de uma aplicação web sem acesso ao código-fonte original  
- Utilizar IA (Gemini no Google AI Studio) para auxiliar no desenvolvimento  
- Aplicar conceitos de Vibecoding e análise de interfaces  
- Refletir sobre o uso de IA no desenvolvimento de software
